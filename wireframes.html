Responsive Design Breakpoints

Mobile (< 768px)

  • Navigation collapses to hamburger menu
  • Grid layouts become single column
  • Contact form and sidebar stack vertically
  • Image slider maintains aspect ratio

Tablet (768px - 1024px)

  • 2-column grid layouts where appropriate
  • Navigation remains horizontal
  • Tables may scroll horizontally if needed

Desktop (> 1024px)

  • Full multi-column layouts
  • Maximum container width of 1200px
  • Hover effects on interactive elements
